# 51-OS-Task-Scheduler
基于51单片机的简易多任务调度器仿真项目

### 实现功能
* 任务状态挂起与运行
* 非阻塞式延时（10ms 系统滴答）
* 双任务并行：0.5秒闪烁与1秒闪烁

### 文件说明
* `main.c`: 核心调度逻辑源码
* `project.pdsprj`: Proteus 8.x 仿真电路图
* `output.hex`: 编译生成的烧录文件
